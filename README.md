# 🚀 Project Overview

Enterprise-grade **Playwright Automation Framework** designed for a **B2C E-Commerce platform** to ensure fast, reliable, and scalable test automation across modern web applications.

This framework focuses on:
- End-to-End UI Automation
- Cross-browser Testing
- API Validation
- Regression Automation
- CI/CD Integration
- Scalable Test Architecture

Built using modern QA engineering practices and optimized for enterprise-level automation execution.

# 📊 Test Execution

Run all tests:

```bash
npx playwright test
```

Run specific browser:

```bash
npx playwright test --project=chromium
```

Run headed mode:

```bash
npx playwright test --headed
```

Generate report:

```bash
npx playwright show-report
```
